# Space Shooter (Dodge the enemy ships)

A simple “space shooter” game that runs in any modern web browser (no build tools).

## How to play

- Open `index.html` in a browser.
- Move your ship with **Arrow keys** or **WASD**.
- Enemy ships spawn at the top and fly downward.
- **Avoid collisions**. If you get hit, it’s game over.

## Controls

- Move: Arrow keys / WASD
- Pause: P
- Restart: R (or the Restart button)

## Files

- `index.html`: UI + canvas
- `styles.css`: styling
- `script.js`: game logic
